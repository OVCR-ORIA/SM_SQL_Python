Running the SM scripts.
Enter substitution variables as follows

Example: 

For the first quarter of calendar year 2014 (STAR METRICS Q1)

beg_date: 01-JAN-2014
end_date: 31-MAR-2014
fsyr: 14 (use 2 digits for fsyr)
period1: 07 use (2 digits for period see below)
period2: 08
period3: 09
coas: 1 (one digit only)

Period	Start Date		End Date
01      01-JUL-2014		31-JUL-2014
02      01-AUG-2014     31-AUG-2014
03      01-SEP-2014     30-SEP-2014
04      01-OCT-2014     31-OCT-2014
05      01-NOV-2014     30-NOV-2014
06      01-DEC-2014     31-DEC-2014
07      01-JAN-2014     31-JAN-2014
08      01-FEB-2014     27-FEB-2014(could be 28-FEB-2014 in a leap year)
09      01-MAR-2014     31-MAR-2014
10      01-APR-2014     30-APR-2014
11      01-MAY-2014     31-MAY-2014
12      01-JUN-2014     30-JUN-2014
